## Prerequisites

- [Astral's **uv** Python project manager][uv]
- [Copier][copier]: `uv tool install copier`

## Usage

```sh
copier copy "gh:smkent/copier-python" /new/project/path
```

[copier]: https://copier.readthedocs.io
[uv]: https://docs.astral.sh/uv/
